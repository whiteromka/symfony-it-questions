# Система публикаций IT контента 

## О проекте

Проект представляет собой систему управления вопросами-ответами в сфере IT.
Каждый пользователь может создать публикацию, задать категорию теги, назначить сложность.
Каждый пользователь может назначить себе набор интересующих его скилов и тем, по которым будет мониторить публикации других пользователей.

## Технологический стек

- **Backend**: PHP 8.3, Symfony 6
- **Database**: PostgreSQL 15
- **Caching**: Redis
- **Web Server**: Nginx
- **Message Broker**: RabbitMQ
- **Containerization**: Docker + Docker Compose

## Развертывание и установка
Клонируйте репозиторий и перейдите в папку проекта

В папке с проектом выполнить команды по очереди:

```bash
# Запускает все сервисы: nginx, PHP-FPM, PostgreSQL, Redis, RabbitMQ в фоновом режиме
docker-compose up -d

# Входим в контейнер с PHP-приложением для выполнения следующих команд
docker-compose exec it-question bash

# Устанавливаем PHP-зависимости через Composer
composer install

# Применяем миграции базы данных для создания таблиц
php bin/console doctrine:migrations:migrate

# Очищаем кеш приложения Symfony
php bin/console cache:clear

# Выходим из контейнера обратно в хост-систему
exit
```
После выполнения этих команд приложение будет доступно по адресу: http://localhost:8080
